# Councel CLI

A Node.js CLI application for managing student counselling data. The application allows counsellors to add, update, remove, search, list and export student data directly from the terminal.

## Features

- Add student records
- Update student information
- Remove student
- Find student
- List students
- Export student data to CSV

## Built With

- Node.js
- MongoDB
- Mongoose
- Commander.js
- Inquirer.js

git clone <repo-url>
cd CLICouncel
